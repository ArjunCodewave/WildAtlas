# iOS Project

## Overview

This project is built using SwiftUI and demonstrates modern iOS development concepts, including multimedia handling, maps integration, animations, and responsive grid layouts.

## Features

* 📷 **AVFoundation** – Used for camera and media-related functionalities.
* 🗺️ **MapKit** – Integrated to display maps and location-based information.
* ✨ **Animations** – Smooth and interactive UI animations to enhance user experience.
* 📱 **Grid View** – Efficiently displays content using `LazyVGrid` for a responsive layout.



## Technologies

* Swift
* SwiftUI
* AVFoundation
* MapKit

## Requirements

* Xcode 16+
* iOS 18+
